*MacOS Only*
1. Install BlackHole 16ch: https://existential.audio/blackhole/
2. In MacOS's Audio MIDI Setup, configure speakers for BlackHole 16ch to 9.1.6 with the channels labeled 1-16 in ascending order
3. In the Downmix Renderer, select BlackHole 16ch as your input device and your 2-channel DAC as your output device.
4. Optionally, adjust the preamp value for your needs
