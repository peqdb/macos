*MacOS Only*
1. Install and unzip Downmix Renderer.zip
3. Install BlackHole 16ch: https://existential.audio/blackhole/
4. In MacOS's Audio MIDI Setup, configure speakers for BlackHole 16ch to 9.1.6 with the channels labeled 1-16 in ascending order
5. Attempt to open Downmix Renderer.app
6. In MacOS System Settings > Privacy and Security > Security
7. In the Downmix Renderer, select BlackHole 16ch as your input device and your 2-channel DAC as your output device
8. Optionally, adjust the preamp value according to your needs
